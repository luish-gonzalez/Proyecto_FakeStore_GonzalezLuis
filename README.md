# FakeStore - Carrito de compras

## Descripción
App web que consume FakeStore API, muestra productos en el DOM y permite agregar al carrito con persistencia en localStorage.

## Funcionalidades
- Consumo de API (productos y categorías)
- Renderizado dinámico (tarjetas)
- Carrito (agregar, aumentar/disminuir, eliminar, vaciar)
- Persistencia con localStorage
- Búsqueda + filtro por categoría + ordenamiento
